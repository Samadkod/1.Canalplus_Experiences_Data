Documentation on Extracting Match Schedules from Flashscore

Objective :

The purpose of this documentation is to provide the necessary information for automating the extraction of live match schedules (including dates, times, home and away teams) for a given season, whether for a league or a European competition (such as the Champions League, Europa League, or Europa Conference League). This extraction is done using the data available on Flashscore via an API.

What is an API?

An API (Application Programming Interface) is a set of definitions and protocols that allows one application to communicate with another. It is used to interact with web services like Flashscore. The data retrieved via an API is often in JSON format (equivalent to a dictionary in Python) or XML.

For example, suppose you want to use Python to retrieve live match schedules for Ligue 1 from Flashscore. You can use the Flashscore API to request this information and integrate it into Snowflake.

API LINKS

To access the match schedules, you need to find the corresponding API link for each league or European competition. Here are the links for some major competitions:

Premier League : "https://api.football-data.org/v4/competitions/PL/matches"
La Liga : "https://api.football-data.org/v4/competitions/PD/matches"
Ligue 1 : "https://api.football-data.org/v4/competitions/FL1/matches"
 Champions League : "https://api.football-data.org/v4/competitions/CL/matches"
Serie A : "https://api.football-data.org/v4/competitions/SA/matches"
Bundesliga : "https://api.football-data.org/v4/competitions/BL1/matches"

Creating an API Account

  1. Go to Football Data.
  2. Click on "Sign Up" and create an account using your professional email.
  3. After creating the account, you will receive an API key via email or directly in your account.
  4. Add this API key to your Python code for data extraction.

Important Considerations for Time Changes

When extracting match times, it is crucial to account for Daylight Saving Time changes to ensure accurate timings. Additionally, you should add 1 hour to Flashscore's UTC times (adding 50 minutes to cover cases where programming might start at 20:55 instead of 21:00, such as on Canal+).

You should also consider the difference between Flashscore's French time and UTC time if using the NAGRA table and most Snowflake tables. Flashscore's French time is UTC+2 during Daylight Saving Time and UTC+1 during standard time.

Note: NAGRA's time is in UTC starting from March 31, 2024, at 2:00 AM. Before this date, the time is recalculated to match French time.
